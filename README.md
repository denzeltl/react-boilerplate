# React Boilerplate

## React Apps

### 1. Install Create React App

#### Standard CRA

```
npx create-react-app my-app
```

#### With TypeScript

```
npx create-react-app my-app --template typescript
```

### 2. Install Material UI and Setup Directories

#### package.json

Add and run script.

```
"setup": "npm install @material-ui/core @material-ui/icons"
```
